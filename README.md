# My-Quiz (Neon Quiz Muhammadiyah)

## Quick start

1. Install dependencies:
   npm install

2. Run dev server:
   npm run dev -- --host

3. Open the URL shown by Vite in your browser or from your phone (same network).

## Add sounds
Place these files into `public/sounds/`:
- click.mp3
- correct.mp3
- wrong.mp3
- bg.mp3 (optional)

You can use any short mp3 files. Browser may block autoplay; background music starts after user interaction (Start).
